# mnemonic

Spaced-repetition flashcards powered by the FSRS algorithm (`ts-fsrs`). Local-first — cards and review history live in your browser via IndexedDB.

## Stack
- React 19 + TypeScript
- FSRS scheduling (`ts-fsrs`)
- Dexie (IndexedDB) persistence
- Vite + Tailwind CSS v4, React Router v7

## Run
```bash
npm install
npm run dev
```
